
# IM-WeiChat
IM-WeiChat---即时通讯(仿微信)

libEaseMobClientSDKLite.a 不包含实时语音功能，libEaseMobClientSDK.a包含所有功能。如果你的 APP 中不需要实时语音功能，使用 libEaseMobClientSDKLite.a 即可。
将环信的 SDK 下载过后拖进项目工程的 lib 文件夹中即可使用;
下载地址 http://www.easemob.com/download/im
